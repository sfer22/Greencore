Diferencia entre URL y URI

URL es hasta el resource, URI es un identificador:

URL:

http://api.example.com/v1/person ----> resource
----   ---  ---------  ---------
proto  host domain     path

URI:

http://api.example.com/v1/person/1478 ----> uid (unique id)
